# ZeckenKiste
書捨て等のスクリプト等を放り込む*ダニ箱*

## 収録物
ゴミの様なダニの様なそんなものです

+ sortSessionManager.pl  
Firefoxのアドオン「SessionManager(https://addons.mozilla.org/ja/firefox/addon/session-manager/)」の.session形式ファイル内のタブを条件を指定して並び替えるスクリプト。

## 使い方

+ sortSessionManager.pl  
多分引数に.session形式のファイルパスを与えてやれば良いかと。
